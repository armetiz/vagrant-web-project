# Vagrant Web Project

There is some default vagrant and puppet files to bootstrap a web project.

You project have to be copied inside **project/** folder. 

## Quick Start

```bash
$ git submodule init
$ git submodule update
$ vagrant up
```

## Requirements

* Vagrant
* VirtualBox
* Internet connection
